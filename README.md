# Computação Gráfica - Projeto 2D

Projeto para a disciplina de Computação Gráfica que demonstra operações simples do OpenGL, além de transformações aplicadas em um objeto.

## Template
A criação desse projeto foi auxiliada por um [template open-source](https://github.com/juliettef/GLFW-CMake-starter). Esta é uma versão modificada
da fonte original, que contém apenas o básico para gerar uma janela.

## Obtendo o código
```
git clone --recursive https://github.com/pandrre/cg-projeto-2d
```

## Build
Para montar a solução em uma máquina Linux, existem alguns scripts auxiliares no projeto.

```bash
git clone --recursive https://github.com/pandrre/cg-projeto-2d
cd cg-projeto-2d
sh ./setup.sh
sh ./run.sh
```

## Comandos

 - Q e W - Rotação.
 - A e S - Aumentar/diminuir.
 - Setas - Translação.
